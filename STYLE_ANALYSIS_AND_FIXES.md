## ОБЗОР ПРОБЛЕМ

После тщательной проверки всех страниц сайта выявлены следующие несоответствия в стилях и структуре:

### 1. ОСНОВНЫЕ ПРОБЛЕМЫ КОНСИСТЕНТНОСТИ

#### a) Дублирование CSS кода
**Проблема:** Каждый HTML файл содержит встроенные стили вместо подключения external style.css
- index.html: встроенные стили (~1000+ строк)
- shop.html: встроенные стили (~1500+ строк)
- equipment.html: встроенные стили
- cosmetic.html: встроенные стили  
- pharma.html: встроенные стили
- cart.html: встроенные стили
- checkout.html: встроенные стили
- order-success.html: встроенные стили

**Решение:** Извлечь все CSS в единый external style.css файл

#### b) Различия в Header (Шапка)
**Проблема:** Header практически идентична во всех файлах, но содержит дублированный код
- Все файлы используют одинаковую структуру
- CSS стили для header встроены во каждый файл
- Нет единой точки для редактирования

**Решение:** Стили header должны быть в external CSS

#### c) Несоответствия в Footer (Подвал)
**Проблема:** Footer имеет разные структуры на разных страницах
- index.html: Footer с 4 колонками + навигационные ссылки
- shop.html: Footer с разными категориями
- Категории (equipment, cosmetic, pharma): Переопределенные footer
- cart.html, checkout.html: Упрощенные footer

**Решение:** Унифицировать footer структуру на всех страницах

## ПОЛНЫЙ СПИСОК ИСПРАВЛЕНИЙ

### ✅ ИСПРАВЛЕНИЕ 1: Унификация Header

**Действие:** Все файлы должны использовать идентичный header

**Текущее состояние:**
- Header код встроен во каждый файл
- Структура: `<header role="banner">` -> `.nav` -> `.logo` -> `.menu` -> `.cart-btn`

**Требуемое состояние:**
- Header стили в external CSS
- HTML структура одинакова на всех страницах
- Hamburger menu работает на всех страницах

### ✅ ИСПРАВЛЕНИЕ 2: Унификация Footer

**Действие:** Создать единый footer для всех страниц

**Требуемая структура:**
```
<footer role="contentinfo">
  <div class="footer-content">
    <!-- 4 колонки с информацией -->
  </div>
  <div class="footer-nav">
    <!-- Навигационные ссылки -->
  </div>
  <div class="footer-bottom">
    <!-- Копирайт и контакты -->
  </div>
</footer>
```

### ✅ ИСПРАВЛЕНИЕ 3: External CSS File

**Действие:** Перенести все CSS в style.css

**Структура style.css:**
1. CSS Reset & Variables
2. Base styles (body, html)
3. Header styles
4. Navigation styles
5. Hero section styles
6. Section styles (about, services, products, etc.)
7. Card styles
8. Shop-specific styles (filters, categories, products)
9. Cart styles
10. Checkout styles
11. Footer styles
12. Responsive design (media queries)
13. Utility classes
