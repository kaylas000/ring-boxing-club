# 🥊 RING BOXING CLUB - Премиум Боксерский Клуб

## Полнофункциональный веб-сайт боксерского клуба

✅ **Проект полностью готов к использованию** — премиум-класс сайта с:
- HTML5 семантической разметкой и SEO оптимизацией
- Адаптивный дизайн (mobile-first, 3 брейкпоинта)
- Полнофункциональная корзина и система заказов
- Интерактивное расписание тренировок
- Система рейтингов и отзывов
- PWA поддержка (работает без интернета)
- REST API backend с Node.js/Express
- Интеграция с платежными системами
- Push-уведомления и email
- Admin-панель для управления контентом

---

## 🚀 Быстрый старт (3 минуты)

### Требования
- Node.js v16+
- Git
- npm

### Установка

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/kaylas000/ring-boxing-club.git
cd ring-boxing-club

# 2. Установите зависимости
npm install

# 3. Запустите сервер
npm start
```

Сайт откроется на **http://localhost:3000**
